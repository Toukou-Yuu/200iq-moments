## 关于本仓库

本仓库系统性地收录了作者在日常生活中所展现的**超凡决策智慧**。

每一条 Case Study 均经过严格的事后复盘，详细记录了：当时无懈可击的逻辑推导过程、现实给出的残酷反馈、以及本可以动动脑子但没有的关键节点。

**本项目长期维护，预计永不停更。**

---

## 快速开始

当你又做了一件蠢事时：

```
1. 深呼吸（可选）
2. 打开本仓库
3. 按模板新建文件：cases/YYYY-MM-DD-一句话标题.md
4. 如实填写，不得为自己开脱
5. git commit -m "feat: 新增一条不可复现的天才决策"
6. 推送，让错误永垂不朽
```
